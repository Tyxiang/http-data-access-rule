# 统计 OVL 结果集

<kbd>格式</kbd>

| 方法 | 格式         | 说明           | 示例          |
| ---- | ------------ | -------------- | ------------- |
| 最大 | `max({key})` | 返回最大项；   | `?s=max(age)` |
| 最小 | `min({key})` | 返回最小项；   | `?s=min(age)` |
| 平均 | `avg({key})` | 返回平均值；   | `?s=avg(age)` |
| 总数 | `count()`    | 返回成员总数； | `?s=count()`  |

<kbd>说明</kbd>

- 对结果集进行统计，返回统计结果；

<kbd>示例</kbd>

```  
?s=max(age)
?s=count()
```
注：以上示例的结果相同。
