# URI Wildcard

<kbd>Format</kbd>

| Wildcard Symbol | Semantic                            |
| :-------------: | ----------------------------------- |
|       `*`       | Match multiple arbitrary characters |
|       `_`       | Match any single character          |
|    `'b,c,d'`    | Match any character among b, c, d   |
|  `'!b,!c,!d'`   | Match any character except b, c, d  |
