# URI 的通配 Wildcard

<kbd>格式</kbd>

|    通配符    | 作用                            |
| :----------: | ------------------------------- |
|     `*`      | 匹配多个任意字符；              |
|     `_`      | 匹配单个任意字符；              |
|  `'b,c,d'`   | 匹配 b、c、d 中任意一个字符；   |
| `'!b,!c,!d'` | 匹配 b、c、d 之外任意一个字符； |
